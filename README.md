# SENHAS-IOT

Esse projeto consiste em um painel de monitoramento integrado a funcionalidades de segurança voltado para o contexto de IoT e Indústria 4.0. Ele inclui um sistema de autenticação que permite o cadastro e login de usuários. O cadastro possibilita a criação de uma conta, enquanto o login autentica o usuário, concedendo acesso ao painel apenas para aqueles que estejam registrados. Ao realizar o login, o usuário tem acesso a um gerador de senhas seguras, que combina letras maiusculas e minúsculas, números e caracteres especiais, facilitando a criação de senhas fortes.
Além da segurança, o painel oferece uma interface de monitoramento IoT que simula dados de sensores industriais, como temperatura e umidade, essenciais em ambientes de Indústria 4.0.
